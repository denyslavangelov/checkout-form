How to Use This Integration

This page demonstrates how the office selection form integrates with Buy Now buttons. When you click a Buy Now button, it will:

  1. Show a modal with city and office selection
  2. Load cities from the Speedy API
  3. Load offices when a city is selected
  4. Create a draft order with the office address
  5. Redirect to Shopify checkout with pre-filled address

Integration Code

To add this to your live site, include this script:

<script src="https://checkout-form-zeta.vercel.app/cdn/office-buy-now-integration.js"></script>

Or add data attributes to your existing Buy Now buttons:

<button data-product-id="YOUR_PRODUCT_ID" data-variant-id="YOUR_VARIANT_ID"> Buy Now </button>

The script will automatically:

  • Find all Buy Now buttons on the page
  • Intercept clicks and show office selection
  • Create draft orders with office addresses
  • Redirect to Shopify checkout
